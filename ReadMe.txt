
使用方法：
  　启动时，会先下载默认网址文件(暂定微软网页)，同时在任务条上创建快捷图标。在精灵身上单击鼠标右键或在任务条图标上单击鼠标右键时，都会弹出相关菜单选项(在任务条上双击鼠标左键，可快速显示或隐藏精灵)，菜单选项中有相应的行为表现方法，包括：变形、休息、行走，和特殊功能＜下载天气预报＞其中，设置下载网址中，可在此天气预报的网址，点下载，稍后，就会显示时新的提示信息。
　　选项中可设置是否静音、顶层显示和是否静止不动等。
　　新增精灵是否全屏行走还是只在任务条上行走选项功能。
    其它包括，关于对话框和是否真的退出功能。

简介：
　运行时，先判断和程序相同路径下有无Tip.txt文件，若无的话，从网站下载文件(当前默认下载为：http://www.microsoft.com网页)，也可在设置下载网址菜单中，将网址改为http://???/1.txt等既可。下载结束后，读取文件，当演示程序停下休息时，会一次在提示窗中显示文件中的一行信息，反覆显示文件内容。其中文件格式为:一行，一行写，一次显示一行，如果要在提示窗中换行的话，可在一行中要换行的地方加上&&符既可。
如下面示例格式：
欢迎您，&&来到未来工作室！
欢迎您，来到未来工作室！
欢迎您，来到未来工作室！
